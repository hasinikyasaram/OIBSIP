# Internship Task: Research Report on Common Network Security Threats

## Project Overview
This repository contains the documentation and research deliverables completed during my Cyber Security internship at **Oasis Infobyte**. The core focus of this task is analyzing the mechanics, impacts, and preventive measures associated with critical network security threat vectors.

### 📋 Deliverables Included
* **`network_security_threats_report.md`**: A detailed, formal research report analyzing core network attacks.

---

## Key Topics Covered

1. **The CIA Triad Framework:** Understanding how network vulnerabilities undermine corporate Confidentiality, Integrity, and Availability.
2. **Volumetric Attacks (DoS & DDoS):** Technical breakdown of request-flooding attacks, botnet orchestration (such as the Mirai Botnet), and strategic defense architectures.
3. **Interception Attacks (MITM):** Examination of data eavesdropping, public Wi-Fi risks, and encryption-based remediation.
4. **Spoofing Technologies:** Exploring the manipulation of protocols across multiple layers, including IP Spoofing, DNS Cache Poisoning, and local ARP Spoofing.
5. **Defense Blueprint:** A compilation of administrative and technical best practices designed to optimize corporate infrastructure security.
