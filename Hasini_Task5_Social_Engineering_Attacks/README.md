# Internship Task: Research Report on Social Engineering Attacks

## Project Overview
This repository hosts the documentation and research artifacts compiled for Task 5 during my Cyber Security internship at **Oasis Infobyte**. The purpose of this research is to deconstruct human-centric attack vectors, examine high-profile corporate security breaches, and outline effective defense methodologies.

### 📋 Deliverables Included
* **`social_engineering_report.md`**: The complete, detailed research report structured with comparative analysis tables and defense strategies.

---

## Key Core Topics Analyzed

1. **Psychological Cyber Vectors:** Breaking down basic Phishing, Spear Phishing, and Executive Whaling techniques.
2. **Impersonation & Lures:** Technical overviews of Pretexting strategies, Baiting traps, and Quid Pro Quo exploits.
3. **Physical Perimeter Defenses:** Mitigating Tailgating risks through employee vigilance and architectural entry control rules.
4. **Historical Historical Breaches:** Comprehensive evaluations of the Target Corporation data breach (2013) and the Twitter Administrative Tool takeover (2020).
5. **Mitigation Blueprints:** Multi-layered security strategies designed to optimize security metrics across individual and organizational levels.
